# US_Visa_Approval
This project is about the US Visa approval. This is a classification problem.


"Create a new conda env":
1. conda create -n venv python=3.12 -y
2. conda activate venv
3. pip install -r requirements.txt [Need to update setup.py before this setup, otherwise will throw error because of -e . and after installation and can see when doing pip list us_visa_approval which is coming from local ]

"Push code to github":
1. git add .
2. git commit -m "First Commit"
3. git push origin main

Steps to be followed in the project.
1) template.py
2) setup.py
3) update requirements.txt
